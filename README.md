## ðŸš€ Getting Started

### Prerequisites
- Web server (Apache/Nginx)
- PHP 7.4 or higher
- MySQL (optional, for database features)

### Installation
1. Clone the repository
2. Configure web server to point to the project directory
3. Update Formspree form ID in contact.php
4. Update business information as needed
5. Upload product images to `assets/img/` directory

### Configuration
- Update contact form endpoint in `contact.php`
- Modify business hours in respective PHP files
- Update social media links in footer sections
- Customize colors in `assets/css/style.css`
